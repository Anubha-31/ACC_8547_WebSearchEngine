The term PHP is an acronym for . PHP is a server-side scripting language designed specifically for web development. PHP can be easily embedded in HTML files and HTML codes can also be written in a PHP file. The thing that differentiates PHP from a client-side language like HTML is, PHP codes are executed on the server whereas HTML codes are directly rendered on the browser.


PHP: Hypertext Preprocessor


It is a simple PHP program to print “Hello world!” message on the screen.
Example:


PHP

<?php
 
/* echo is a print command */
echo "Hello world!"; 

?>


Output:
Hello world!
